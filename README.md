# Smart Travel

Smart Travel is a travel planning web application created to help users organize trips in a simple, practical, and user friendly way. The platform is designed to centralize important travel information, making it easier to explore destinations, plan activities, and manage essential trip details in one place.

This project was developed as a final project and demonstrates front end web development concepts using HTML, CSS, and JavaScript. Its purpose is to provide a smoother travel planning experience while also showing clean structure, usability, and functionality in a real world style application.
